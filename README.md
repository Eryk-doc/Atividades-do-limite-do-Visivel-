# Atividades — Códigos

Repositório para armazenar os códigos das minhas atividades.

Estrutura sugerida:

- `exercicios/` — códigos organizados por disciplina ou tópico
- `projetos/` — pequenos projetos e exercícios maiores
- `notas/` — anotações, enunciados e referências

Como iniciar um repositório Git local e enviar para o GitHub (opcional):

1. Inicializar o repositório local:

```powershell
cd "c:\Users\erykm\Documents\Atividades python"
git init
git add .
git commit -m "Initial commit"
git branch -M main
```

2. Criar um repositório no GitHub (pela web ou com `gh`) e adicionar o remoto:

```powershell
git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPO.git
git push -u origin main
```

Substitua `SEU_USUARIO` e `NOME_DO_REPO` pelos valores corretos.

Se quiser, eu posso criar o repositório remoto para você (preciso do seu ok e de autorização).
